**Q1. What is a function in Python?**
It is basically a block of code that performs a specific task.

**Q2. Why are functions important?**
They are important because they help avoid repeating code and make the program easy to understand.

**Q3. What is difference between print and return?**
Print is just used to show the output, but return actually sends the value back to the program.

**Q4. What are parameters in function?**
Parameters are simply the inputs that we give to the function to work with.

**Q5. Can a function run without parameters?**
Yes, a function can definitely run without parameters, just like the `greet()` example.

Summary of learnings:
* A function is a block of code used to perform a task and reuse code.
* We create them using the `def` keyword.
* They make the program much easy to understand by breaking it into small parts.
* We can pass inputs called parameters, and get values back using the `return` statement.
