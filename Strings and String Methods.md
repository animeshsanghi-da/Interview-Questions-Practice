**Q1. What is a string in Python?**
A string is a sequence of characters used to represent text, enclosed in quotes.

**Q2. What is string indexing?**
Indexing refers to the specific position of each character, starting at 0 for the first letter.

**Q3. What is string slicing?**
Slicing is the process of getting a sub-section of a string by defining a start and end index.

**Q4. Name some common string methods.**
Common methods include `upper()`, `lower()`, `strip()`, and `replace()`.

**Q5. What does replace() do?**
The `replace()` method swaps out a specific part of the string with another piece of text.

Summary of learnings:
* Strings are the primary way to handle text data in Python.
* You can access specific characters or parts of a string using indexing and slicing.
* Built-in methods allow for easy transformation, such as changing case or removing whitespace.
