**Q1. What is a list in Python?**
It's basically a collection of items that are written inside square brackets.

**Q2. How do you add item to a list?**
You can easily add items using the append() method or the insert() method.

**Q3. What is difference between append() and insert()?**
append() adds the item at the very end of the list, while insert() adds the item at a specific position you choose.

**Q4. How to remove item from list?**
You can use remove() to delete a specific item, or use pop() if you just want to remove the last item.

**Q5. Can a list store different data types?**
Yes, a list can store different types of data like numbers, text, or mixed data all together.

Summary of learnings:
* A list is just a collection of items written inside square brackets `[]`.
* They can store mixed data like numbers and text together.
* Each item gets an index and the first item always starts from 0.
* We can use different methods like `append()` to add, `remove()` to delete, and `sort()` to order the items.
