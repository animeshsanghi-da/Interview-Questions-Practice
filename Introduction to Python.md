**Q1: What is Python?**
Python is a programming language used to give instructions to a computer. It is used to write programs and solve problems.

**Q2: Why is Python easy to learn?**
Python is easy to learn because its code is simple and readable, and it requires writing less code compared to many other languages.

**Q3: Who created Python?**
Python was created by Guido van Rossum in 1991.

**Q4: Where is Python used?**
Python is widely used in areas like web development, data science, machine learning and AI, automation, and software development.

**Q5: What is a variable in Python?**
A variable is used to store values in a program.

- **Python Basics:** Learned that Python is a beginner-friendly programming language created in 1991, known for its simple and readable syntax.
- **Real-World Uses:** Discovered its wide range of applications across different fields, including web development, data science, machine learning, and automation.
- **Core Concepts:** Understood basic programming elements, such as how to use variables to store data and the `print()` function to display output.
