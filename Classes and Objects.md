**Q1. What is a class?**
It is basically a blueprint or template that we use for creating objects.

**Q2. What is an object?**
An object is an instance of a class that represents a real-world item in our code.

**Q3. What does __init__ method do?**
It is a special method that automatically runs when a new object is created to initialize its attributes.

**Q4. What is self in Python?**
The 'self' keyword refers to the current object being created or used, allowing us to access its variables.

**Q5. Can we create multiple objects from one class?**
Yes, just like a single blueprint can build many houses, we can create as many objects as we need from one class.

**Summary**:
* A class acts as a blueprint, while an object is a real-world instance created from that class. 
* We use the __init__ method to initialize attributes, and 'self' helps us access the specific object's data.
