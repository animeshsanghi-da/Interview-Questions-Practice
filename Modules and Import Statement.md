**Q1. What is a module in Python?**
A module is a file containing Python code that can be reused in other programs.

**Q2. Why do we use modules?**
We use them to organize our code, avoid repetition, and access pre-written features.

**Q3. What is difference between import and from import?**
`import` loads the whole module, while `from...import` only loads the specific parts you need.

**Q4. What is alias in import statement?**
An alias is a nickname (like `m` for `math`) that makes it faster to refer to a module.

**Q5. Name some built-in Python modules.**
Examples include `math` for calculations, `random` for random numbers, and `datetime` for dates.

Summary of learnings:
* Modules allow programmers to break large programs into smaller, manageable files.
* The `import` statement is used to bring external code into your current file.
* Using aliases and specific imports helps keep code clean and efficient.
