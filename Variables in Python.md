**Q1. What is a variable in Python?**
It's basically like a container that holds a value or data in the program's memory.

**Q2. Why do we use variables in programming?**
We use them to store info so we can use the same value multiple times or do calculations without writing the numbers again and again.

**Q3. What are the rules for naming variables?**
They must start with a letter or underscore, cant start with a number, and you shouldn't use spaces or special characters.

**Q4. What types of data can variables store?**
They can store different types of data like whole numbers (int), decimals (float), text (string), or true/false values (bool).

**Q5. Can the value of a variable change during execution?**
Yes, the value stored in a variable can be easily used and changed while the program is running.

Summary of learnings:
* Variables act as memory containers to store and organize data.
* Valid names must start with letters or `_` and can't have spaces or special symbols.
* Variables help make the code easy to understand and lets us reuse values for calculations.
* The main data types you can store are integers, floats, strings, and Booleans.
