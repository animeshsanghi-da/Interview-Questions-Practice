**Q1. What is function in Python?**
It is basically a block of code that performs a specific task and helps in code reuse.

**Q2. Difference between parameter and argument?**
Parameters are the variables defined inside the parentheses when creating the function, while arguments are the actual values passed to the function when calling it.

**Q3. What is return keyword?**
The return keyword is used to send a result or value back from the function to the main program.

**Q4. Why we use functions?**
We use them to avoid repeating code, make the program easier to understand, simplify debugging, and save time.

**Q5. Can one function call another function?**
Yes, one function can call another function from inside its own code block.

Summary of learnings:
* A function is a block of code used to perform a task and reduce repetition.
* We create them using the `def` keyword and run them by calling their name.
* Functions make programs clean and organized, allowing inputs via parameters and outputs via the `return` keyword.
