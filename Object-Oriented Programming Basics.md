**Q1. What is OOP?**
Object-Oriented Programming (OOP) is a way of writing and structuring programs using classes and objects.

**Q2. What is difference between class and object?**
A class is the underlying blueprint, whereas the object is the actual, tangible instance built from that blueprint.

**Q3. What is constructor?**
A constructor (like __init__) is a special method that automatically sets up and initializes a new object upon creation.

**Q4. What are OOP features?**
The four main features are Encapsulation (grouping data/methods), Inheritance (reusing code), Polymorphism (same method, different use), and Abstraction (hiding details).

**Q5. Why is OOP useful?**
It is incredibly useful because it helps us organize complex code better, reuse existing code easily, and model real-world problems effectively.

**Summary**:
* OOP structures our code around objects rather than just functions, making it cleaner and more organized.
* By leveraging features like inheritance and encapsulation, we can write reusable and secure code.
