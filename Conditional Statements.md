**Q1. What is the purpose of conditional statements?**
They are used to make decisions in a program by running different code based on certain conditions.

**Q2. What is the difference between if and if–else?**
The 'if' statement executes code only if the condition is True, while 'if-else' runs one block if the condition is True, and a completely different block if it is False.

**Q3. When do we use elif?**
We use 'elif' when there are multiple different conditions to check one after the other.

**Q4. What is a nested if statement?**
It is simply an 'if' statement written inside another 'if' statement to check a condition within a condition.

**Q5. Can we use multiple elif blocks?**
Yes, we can use multiple 'elif' blocks to check as many different conditions as needed.

Summary of learnings:
* Conditional statements are used for decision-making in a program.
* We use `if` to check a single condition, while `if-else` handles both True and False outcomes.
* We can use `elif` for multiple conditions, and nested `if` allows checking conditions inside other conditions.
