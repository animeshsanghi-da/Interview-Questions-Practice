**Q1. What is Data Science?**
It's basically the process of studying data to find patterns and useful info. It uses stats and programming to solve real problems and help make better decisions.

**Q2. What are the main steps in a data science project?**
The main steps are collecting data, cleaning it to fix errors, analyzing it with algorithms, and then visualizing the results. Finaly, you use those insights for decision making.

**Q3. Which tools are commonly used in Data Science?**
People mostly use Python, SQL, and R. For specific tasks, tools like Pandas for analysis and Tableau or Power BI for dashboards are super common.

**Q4. What is the difference between data analysis and data science?**
Data Science is the whole field that combines programming and math to solve problems. Data analysis is just one part of the process where you study the data to find insights.

**Q5. Can you give a real-world example where Data Science is used?**
A good one is e-commerce sites giving product recommendations based on what you searched. Banks also use it to flag weird transactions for fraud detection.
