Q1. **What is Google Colab and how is it different from Jupyter Notebook?**
It's a cloud-based notebook tool by Google that runs right in your browser. It is very similar to Jupyter but the best part is it doesn't need any installation on your computer.

Q2. **How do you open and create a notebook in Colab?**
You can go directly to colab.research.google.com and click new notebook, or you can just type colab.new in your browser URL. You can also open it from your Google Drive.

Q3. **How can you enable GPU in Google Colab?**
Just click on 'Runtime' in the top menu, select 'Change runtime type', choose 'GPU', and hit save.

Q4. **Explain code cell and text cell.**
A code cell is where you actually write and execute your Python code. A text cell is used to write notes or explanations and it supports markdown.

Q5. **What are advantages of using Google Colab?**
The main benefits are that there is no setup needed, you get free access to GPUs and TPUs, and it automatically saves your work to Google Drive.

Summary of learnings:
* Google Colab is a free cloud tool where you can write and run python code in your browser without downloading anything.
* It is great for data scientists and students because it gives free access to GPU for faster machine learning.
* The interface has Code Cells for running scripts and Text Cells for writing markdown notes.
* Everything auto-saves to Google Drive so its super easy to share notebooks with others.
