**Q1. What is list comprehension?**
It is a concise, one-line way to create lists in Python.

**Q2. How is it different from normal loop?**
It combines the loop and optional conditions into a single line, making it shorter and easier to read.

**Q3. Can we use condition in list comprehension?**
Yes, you can add an `if` statement at the end to filter which items get added to the list.

**Q4. Is list comprehension faster than loop?**
Yes, it is generally more efficient and faster than building a list with a standard for loop.

**Q5. Where is list comprehension useful?**
It is perfect for simple transformations, like squaring numbers or changing the case of string elements in a list.

Summary of learnings:
* List comprehension is a short, efficient way to generate lists.
* It uses the syntax: `[expression for item in sequence]`.
* It makes the code cleaner and runs faster than traditional loops.
