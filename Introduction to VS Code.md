Q1. **What is VS Code?**
It's basically a free software made by Microsoft that we use to write code for different languages.

Q2. **Why is VS Code used?**
People use it because it is very fast, easy for beginners, and supports a lot of programing languages.

Q3. **How do you run Python code in VS Code?**
You just need to install the python extension, write your code, and then click the Run button or run it in the terminal.

Q4. **What is extension in VS Code?**
Extensions are like extra features or add-ons you can install to make the code editor do more things.

Q5. **What is terminal in VS Code?**
It's a screen at the bottom of the editor where you can actually run your code or type commands.

Summary of learnings:
* VS Code is a free code editor by Microsoft used to write programs.
* It is really popular because its fast and easy for beginners.
* You can open folders, create files, and run code right from the terminal at the bottom.
* We can install extensions to add new features like Python support or code formatting.
