**Q1. What is structured data?**
It is data that is nicely organized in rows and columns so its easy to search and process. 

**Q2. Give examples of unstructured data.**
Some common examples are photos, videos, emails and CCTV camera footages.

**Q3. What makes semi-structured data different from structured data?**
It doesn't have a strict table format like structured data, but instead uses tags or keys to organize things.

**Q4. Why is unstructured data difficult to analyse?**
Because it dosen't follow any fixed format or rows, so we need special AI or computer vision to process it.

**Q5. How can AI analyze CCTV footage data?**
AI uses computer vision models to detect things like theft, suspicious activity, or how customers are moving in the video.

Summary of learnings:
* Data is just raw info like text or numbers used by companies to make decisions.
* Structured data is saved in fixed rows and coloumns like Excel sheets or SQL.
* Unstructured data has no fixed format (like images or videos) and is hard to analyse without AI.
* Semi-structured data is partly organized using tags or keys, like JSON files.
