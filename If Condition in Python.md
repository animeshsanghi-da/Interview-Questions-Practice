Q1. **What is an if condition in Python?**
It is used to make decisions in a python program by checking if a situation is True or False. 

Q2. **What is the difference between if, elif, and else?**
'if' checks the main condition, 'elif' checks another condition if the first one fails, and 'else' is what runs when no conditions are true.

Q3. **Can we write an if statement without else?**
Yes, you can write a simple if statement all by itself without needing an else block.

Q4. **Where are if conditions used in real-world programs?**
They are used in places like ATM machines to check balance, login systems to verify passwords, or for weather alerts.

Q5. **How does Python evaluate conditions?**
Python checks the condition to see if it's True or False, and if it is True, it executes the code inside that block.

Summary of learnings:
* If conditions let a program make decisions based on different situations.
* The three main types we use are if, if else, and if elif else.
* The code inside the block only runs if the condition is True, otherwise it gets skipped.
* We use them for real life things like checking if a student passes based on marks.
