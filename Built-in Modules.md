**Q1. What are built-in modules?**
They are ready-made sets of functions that come pre-installed with Python, meaning we do not need to download them separately to use them.

**Q2. How do you use math module?**
We import it to perform mathematical calculations, like finding the square root of a number using math.sqrt() or fetching the value of pi.

**Q3. What does random module do?**
The random module helps us generate random numbers or pick a random item from a given sequence or list.

**Q4. How to get current date in Python?**
We can get the current date by importing the datetime module and calling datetime.date.today().

**Q5. What is sys module used for?**
The sys module is used for system-related operations, such as checking the platform we are on or finding out which version of Python is currently running.

**Summary:**
* Built-in modules provide a huge collection of pre-written functions that save us development time.
* Commonly used modules include 'math' for complex calculations, 'random' for generating random values, 'datetime' for time and date tracking, and 'sys' for system-level information.
