**Q1. What is the purpose of break statement?**
It is used to stop the loop immediately when a specific condition is met.

**Q2. How is continue different from break?**
While break stops the entire loop, continue only skips the current iteration and jumps to the next round.

**Q3. When do we use pass?**
We use the pass statement as a placeholder when code is required by syntax but we don't want to execute anything yet.

**Q4. Can break be used outside loop?**
No, it is specifically a control statement designed to be used inside loop structures.

**Q5. What happens after continue is executed?**
The program immediately moves to the start of the next loop cycle, ignoring any code written below the continue statement.

Summary of learnings:
* Break, continue, and pass are tools to control how a loop behaves.
* `break` exits the loop, `continue` skips the current step, and `pass` does nothing.
* These statements help manage the flow of logic within repetitive tasks.
